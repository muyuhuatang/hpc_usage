Operation:

Usage

  1. open electerm, and connect to the luddy server;

  2. module load python;

  3. change your working direction: cd ./WORKINGDIRECTION;

  4. jupyter-lab --no-browser --port=8888;

  5. check the message about where the jupyter running, especially the port that used. For example, you may see such a message:

    The jupyter server is running at:
http://127.0.0.1:<PORT>
http://127.0.0.1:<PORT>?tocken=<TOKEN>

  6. copy the token from the message that you saw in last step;

  7. open your terminal of your own laptop, and input ssh -N -f -L localhost:<PORT>:localhost:<PORT>:YOURIUUSERNAME@LUDDY SERVER NAME.luddy.indiana.edu;

  8. open your brower, open the link:http://localhost:<PORT>, paste your token and log in;

Log out

  1. close that page of the brower;

  2. in your own laptop, kill the job by

    sudo lsof -i -P | grep LISTEN | grep :<PORT>
sudo kill -9 <PID>

    The <PID> is the number in the second column of the message when you input the first command line.

  3. in Luddy server, type control+c, and then input y to shut down the server
